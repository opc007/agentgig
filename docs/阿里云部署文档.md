## 项目部署结构

```
/opt/
├── agentgig/                  # AgentGig 项目（独立目录）
│   ├── backend/               # FastAPI 后端
│   │   ├── venv/              # Python 3.11 虚拟环境
│   │   ├── main.py            # 入口
│   │   ├── .env               # 环境变量（含 LLM API Key）
│   │   └── agentgig.db        # SQLite 数据库
│   ├── frontend/              # React 前端
│   │   ├── dist/              # 构建产物
│   │   └── src/
│   ├── sdk/
│   ├── cli/
│   ├── docs/
│   └── templates/
│
├── gateway-mvp/               # 聚合算力平台（不要动）
└── gateway-mvp-backups/       # gateway 备份
```

## 服务端口

| 服务 | 端口 | 说明 |
|------|------|------|
| Nginx | 80 | 统一入口，按域名分流 |
| AgentGig 后端 | 8001 | FastAPI (uvicorn) |
| AgentGig 前端 | - | 由 Nginx 代理静态文件 |
| Gateway-MVP | 8081 | 聚合算力平台 |
| Nginx 备用 | 8080 | 直接访问 AgentGig（不经过域名） |

## Nginx 分流规则

```
80 端口
├── Host: agentgig.ainn.asia  → AgentGig (localhost:8001 + 静态文件)
└── 其他域名/IP               → Gateway-MVP (localhost:8081)
```

配置文件：`/etc/nginx/conf.d/agentgig.conf`

## Systemd 服务

### AgentGig 后端
```
服务名：agentgig.service
配置文件：/etc/systemd/system/agentgig.service
状态命令：systemctl status agentgig
重启命令：systemctl restart agentgig
日志命令：journalctl -u agentgig -f
```

### Gateway-MVP
```
服务名：gateway-mvp.service
配置文件：/opt/gateway-mvp/.env
状态命令：systemctl status gateway-mvp
重启命令：systemctl restart gateway-mvp
```

### Nginx
```
服务名：nginx.service
主配置：/etc/nginx/nginx.conf
AgentGig 配置：/etc/nginx/conf.d/agentgig.conf
测试配置：nginx -t
重载配置：nginx -s reload
```

## 更新部署流程

### 更新后端代码
```bash
# 1. 本地同步代码到服务器
rsync -avz --exclude 'node_modules' --exclude 'venv' --exclude '.git' \
  --exclude '__pycache__' --exclude '*.pyc' --exclude 'agentgig.db' \
  ./backend/ root@8.163.54.123:/opt/agentgig/backend/

# 2. 服务器上重启服务
ssh -i ~/.ssh/boss_assistant_aliyun root@8.163.54.123 \
  "systemctl restart agentgig"
```

### 更新前端代码
```bash
# 1. 本地同步代码
rsync -avz --exclude 'node_modules' --exclude 'dist' \
  ./frontend/ root@8.163.54.123:/opt/agentgig/frontend/

# 2. 服务器上构建
ssh -i ~/.ssh/boss_assistant_aliyun root@8.163.54.123 \
  "cd /opt/agentgig/frontend && npm install && npm run build"
```

### 一键全部更新
```bash
rsync -avz --exclude 'node_modules' --exclude 'venv' --exclude '.git' \
  --exclude '__pycache__' --exclude '*.pyc' --exclude 'agentgig.db' \
  --exclude 'dist' \
  ./ root@8.163.54.123:/opt/agentgig/

ssh -i ~/.ssh/boss_assistant_aliyun root@8.163.54.123 \
  "cd /opt/agentgig/frontend && npm install && npm run build && \
   systemctl restart agentgig"
```

## LLM 配置

AgentGig 接入了 MiniMax M3 做智能需求对话。

配置文件：`/opt/agentgig/backend/.env`
```
LLM_BASE_URL=https://api.minimaxi.com
LLM_API_KEY=<MiniMax API Key>
LLM_MODEL=MiniMax-M3
```

修改后重启：`systemctl restart agentgig`
